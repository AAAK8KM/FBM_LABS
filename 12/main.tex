\section{Вычисление числа Маха двумя способами}

В данном отчёте рассматриваются два итерационных метода расчёта числа Маха:
\begin{itemize}
    \item через отношение площадей (метод 1)
    \item через отношение давлений (метод 2)
\end{itemize}

Для расчётов принимается показатель адиабаты $\gamma = 1.4$ (воздух).

\begin{enumerate}
    \item \textbf{ Расчёт через отношение площадей}


Итерационная формула для расчёта числа Маха через отношение площадей имеет вид:

\begin{equation}
M_i = \left[
\frac{\gamma+1}{\gamma-1}
\left(\frac{A}{A^*} M_{i-1}\right)^{\frac{2(\gamma-1)}{\gamma+1}}
- \frac{2}{\gamma-1}
\right]^{1/2}
\end{equation}

где:
\begin{itemize}
    \item $M_i$ --- число Маха на $i$-й итерации
    \item $M_{i-1}$ --- число Маха на предыдущей итерации
    \item $A/A^*$ --- отношение площади сечения к критической площади
    \item $\gamma = 1.4$ --- показатель адиабаты
\end{itemize}


\begin{lstlisting}[language=Python, caption={Расчёт числа Маха через отношение площадей}]
gm = 1.4
f = float(input())  # Отношение площадей A/A*
M, Mn = 2, 0
merr = 0.005

while M - Mn > merr:
    Mn = M
    M = ((gm+1)/(gm-1) * (f*M)**(2*(gm-1)/(gm+1)) - 2/(gm-1))**0.5
    print(M, M-Mn)
\end{lstlisting}


\begin{enumerate}
    \item Задаётся начальное приближение $M_0 = 2$
    \item Итерационный процесс продолжается до достижения точности $\varepsilon = 0.005$
    \item На каждой итерации вычисляется новое значение числа Маха по формуле (1)
    \item Процесс завершается, когда $|M_i - M_{i-1}| \leq \varepsilon$
\end{enumerate}

\item \textbf{ Расчёт через отношение давлений}

Итерационная формула для расчёта числа Маха через отношение давлений:

\begin{equation}
M_i^2 = \frac{\gamma-1}{2\gamma} + \frac{\gamma+1}{2\gamma}
\left(\frac{P_0'}{P_0}\right)^{-\gamma+1}
\left[
\frac{2}{\gamma+1}
\left(\frac{1}{M_{i-1}^2} + \frac{\gamma-1}{2}\right)
\right]^{-\gamma}
\end{equation}

где:
\begin{itemize}
    \item $M_i$ --- число Маха на $i$-й итерации
    \item $P_0'/P_0$ --- отношение давлений
    \item $\gamma = 1.4$ --- показатель адиабаты
\end{itemize}


\begin{lstlisting}[language=Python, caption={Расчёт числа Маха через отношение давлений}]
gm = 1.4
p1, p2 = map(float, input().split())  # Отношение давлений
M, Ms = 4, 1
n = 10
i = 0

while abs(M - Ms) > 0.00001 or i < n:
    Ms = M
    M = (gm-1)/(2*gm) + (gm+1)/(2*gm) * (p2/p1)**(1-gm) * \
        (2/(gm+1)*(1/Ms**2 + (gm-1)/2))**(-gm)
    print(M**0.5, M-Ms)
    i += 1
\end{lstlisting}


\begin{enumerate}
    \item Задаётся начальное приближение $M_0 = 4$
    \item Максимальное число итераций: $n = 10$
    \item Итерационный процесс продолжается до достижения точности $\varepsilon = 10^{-5}$ или превышения максимального числа итераций
    \item На каждой итерации вычисляется новое значение $M_i^2$, затем извлекается квадратный корень
    \item Процесс завершается, когда $|M_i - M_{i-1}| \leq \varepsilon$ или $i \geq n$
\end{enumerate}
\end{enumerate}

\section{ Обсуждение результатов и выводы}
В ходе данной лабораторной работы были изучены течения с недорасширением, перерасширением, а также близкие к расчетным течения через сопа Лаваля.
Полученные в результате эксперимента геометрические числа Маха с хорошей точностью
совпадают с аналитически вычисленными значениями.
Коэффициенты расчетности для каждого режима посчитаны и приведены вместе с картин-
ками свехзвукового истечения для каждого из исследуемых сопел.
Давления в ресивере, рассчитанные теоретически, для близкого к расчетному режиму, которые были взяты в качестве опорных в эксперименте, хорошо демонстрируют картинки расчетного режима.
